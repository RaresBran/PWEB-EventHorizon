# EventHorizon

A Java + Angular web application for discovering events in your city and saving them to your personal list.

## Features

- Browse local events
- Add events to your favorites
- Java Spring Boot backend
- Angular frontend

## Tech Stack

- **Backend:** Java, Spring Boot
- **Frontend:** Angular
- **Build Tool:** Gradle

## Getting Started

### Backend

```bash
cd Backend
./gradlew bootRun
```

### Frontend

```bash
cd Frontend
npm install
ng serve
```
